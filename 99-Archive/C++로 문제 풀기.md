# 어떤 방식으로 풀어야 하는가?

1. 일단 문제를 보자
2. 예제를 확인
3. "일단 풀자"
4. 풀고 최적화
5. 검토하기
6. 구현하기
7. 테스트

# 하면 좋은 것들

1. 직접 풀자 - 힌트는 최대한 보지 말고
2. 정말 필요한 정보만 - 답이 아닌 과정에 대한 힌트는 ㄱㅊ
3. 디버깅 + 테스트는 필수
4. 다른 사람의 풀이도 참고하고 얻을 게 있으면 배워갈 것

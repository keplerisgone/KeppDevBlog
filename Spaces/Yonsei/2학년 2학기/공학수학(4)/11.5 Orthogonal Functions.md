Orthogonal functions

## Orthogonal functions

어떠한 interval $a\leqq x\leqq b$﻿ 에서 정의된 $y_1(x),y_2(x),y_3(x)...$﻿ 에 대해 다음이 성립하면 해당 function들은 ==orthogonal with respect== ==$r(x)$==﻿ ==on the interval== ==$a\leq x\leq b$==﻿ 하다고 합니다.

> $(y_m,y_n)=\int_a^b r(x)y_m(x)y_n(x)dx =0 \ \ (m\neq n)$﻿

여기서 $r(x)$﻿ 은 ==weight function== 이라고 하고, $(y_m,y_n)$﻿은 standard notation이라고 합니다.

함수 $y_m$﻿ 의 ==norm==, $||y_m||$﻿ 은 다음과 같이 정의됩니다.

> $||y_m|| = \sqrt{(y_m,y_n)}=\sqrt{\int_a^br(x)y_m^2(x)dx}$﻿

그리고 함수 $y_1, ...,y_n$﻿ 의 orthonormal 은 다음과 같이 정의됩니다.

> $(y_m,y_n)=\int_a^b r(x)y_m(x)y_n(x)dx =\delta_{mn},$﻿  
>   
> $= 0 $﻿ if $m\neq n$﻿  
>   
> $= 1$﻿ if $m=n$﻿

풀어서 설명하면, m과 n 이 다르면 내적 값이 0이 나오고, m과 n이 같으면 1이 나옵니다. 이는 각 함수의 norm이 1이라는 뜻입니다.

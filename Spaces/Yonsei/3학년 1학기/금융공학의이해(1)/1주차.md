## Derivative (파생상품)

- *기초자산*(underlying asset) : 주식, 채권, 외환, 상품 등등 대부분의 거래
- 기초자산의 가치에 의해 자기자신의 가치가 변하는 상품
	- **forward**(선도계약)/**future**(선물)
		- 미래 시점에 미래의 물건을 얼마에 팔자!
		- 기초자산이 정해져 있다 - maturing, delivery price
		- long (buy), short (sell) position
		- payoff diagram - 두 position이 얼마나 손해/이익을 보는가
		- forward - OTC, future - exchange (조건 제약)
		- 그래프 - 만기시점의 금액 (St), payoff가 y축
	- **option**(옵션)
		- call / put
- 왜 거래하는가? -> 리스크를 전가하기 위해서
	- 왜 사는가? -> 그만큼 보상을 주니까
- **거래방법**
	- 거래소 (exchange) : 장내 시장
	- **OTC** (over the counter) : 장외 시장 <- 더 넓다
### OTC
- 2008 금융 위기 이후 제도의 변화
1. CCP (Centural Counter Policy) : 중심의 무언가를 이용해서만 거래 가능
2. Margin, Collateral : 담보를 꼭 맡겨라
3. Standard, Repost : 표준화, 보고해라
### Option
- 살수 있는(call) 팔수 있는(put) 권리 - 이득을 보는, long
	- 반대는 의무가 됨 - 손해를 봐야 하는, short
- 얘는 그래프를 둘로 나눠서 생각해야함 의무/권리
- 권리를 사는 것이기 때문에 옵션 자체의 가격이 따로 있음
	- forward / future는 애초에 '가격을 정해놓고' 약속하는 것


# 옵시디언을 어떻게 정리할 것인가?

옵시디언은 물론 좋은 노트 앱이다. 

방향성을 세우자.
- 모바일에서도 가볍게 사용이 가능하도록
	- 없으면 안되는 플러그인만 설치하자.
- 일원화, 노트 정리를 모두 여기서 처리한다. 
> [!question] 
> 손글씨는 어떻게 처리할 수 있는가?

손글씨 같은 경우는 노트 앱 이용 + 스크린샷 밖에 답이 없는 것 같다.

> [!question]
> pdf 필기는 어떻게 처리할까?

pdf 손글씨는 옵시디언 만으로 처리할 수가 없다.
차라리 노트 작성 방향성을 달리하는 게 낫지 않을까??
수식은 LeTex 플러그인이 이쁜게 많아 다행이다.

- 노트 정리법 공부
	- 대체 저 많은 노트를 어떻게 분류할 것인가
- 미디어 사용은 어떻게 하지
	- pdf는 언젠가 사용할 때가 올텐데.. 스크린샷으로만 충당하기는 어렵다.
	- pdf를 옵시디언 내부에 저장 + 아이패드 미리보기로 필기 + better export pdf plugin 조합을 사용하면 빠르게 동기화는 가능하다. 
	- 이미지의 경우 imgur을 사용 중이지만 외부에 의존하는게 좋다고 생각하지는 않는다.
	- 사실 노트를 쓸 때 미디어를 많이 쓰고 싶지는 않다. 텍스트 기반으로 하고 싶지...
	
# 하고자 하는 것

우선 옵시디언 노트를 개편한다. Linter 플러그인을 이용해서 노트 형식을 통일화, 각종 문법을 적재적소에 활용하는 법을 세울 예정이다. 이런 상황에서는 이렇게 써라 같은...
사용하고자 하는 플러그인 정리, 옵시디언 공부, 마침 좋은 자료가 있길래 첨부한다.
https://kaminik.tistory.com/
이것도 옵시디언 문법을 몰라서 이쁘게 못 넣는다
노트 쓰는 법을 정한다. 경우의 수는 다음과 같다.
	각종 강의를 들을 때, 이거는 순서가 정해져 있기 때문에 나보고 어쩌라고다.
	pdf 이용한 강의들, 이거는 어떡하냐 진짜 pdf에다가 필기를 하고 싶긴한데 (무조건 손글씨 필요한 경우가 대다수) pdf를 여기다가 다 불러올 수도 없고 이거는 방법을 더 찾아보아야 한다
	그냥 정보 정리글은 쉽다. 블로그처럼 쓴다.
